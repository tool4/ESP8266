Activity (motion) monitor project with remote upload of data logs to web server and nice presentation (using google charts) of hourly activity for every day - see sample_chart.png for reference.

trivial connection:
- PIR sensor out connected to ESP8266 D7
- PIR sensor VCC connected to ESP8266 5V
- PIR sensor GND connected to ESP8266 GND

www files should be placed on accesible web serwer with php (e.g. local raspberry + apache2 + php)
